Pandoc-ModernCV is a pandoc fa­cil­i­ties for type­set­ting mod­ern cur­ricu­lums vi­tae. Inspired by the well known Latex ModernCV, it is fairly cus­tomiz­able, al­low­ing you to use predefined themes and define your own style by changing colors, fonts, etc.

Pandoc-ModernCV currently support pdf and html5 export formats. The html5 output is responsive and support rendering for small to large screens.

Getting Started

The simplest way to get started with pandoc-moderncv is to use the provided scaffold. In a terminal just do:

$ make scaffold
$ make html

What it does it that it creates a scaffold cv located in the /cv directory and builds an html version of it. To open the generated cv just do:

$ open dist/cv.html

There you are!
