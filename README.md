pandoc-moderncv
===============

Pandoc fa­cil­i­ties for type­set­ting mod­ern cur­ricu­lums vi­tae. Inspired by the well known Latex ModernCV, it is fairly cus­tomiz­able, al­low­ing you to use predifined theme and de­fine your own style by chang­ing the colours, the fonts, etc.

# Getting Started

The simplest way to get started with *pandoc-moderncv* is to use the provided scaffold. In a terminal just do:

    $ make scaffold
    $ make html

What it does it that it creates a scaffold *cv* located in the /cv directory and builds an html version of it. To open the generated cv just do:

    $ open dist/cv.html

There you are!
